In this class we'll prove the invariance of Euler characteristic for surfaces and understand the classification theorem. We'll see how this innocent invariant leads to non-trivial results like the Sperner's lemma and Brouwer's fixed point theorem.

Keywords: classification of surfaces (statement), Euler characteristic, genus, Sperner's lemma, Brouwer's fixed point theorem.

Prereqs: None
Homework: Required
